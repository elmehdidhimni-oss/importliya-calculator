الشحن الدولي

نوع الشحن، الوزن، الحجم والتأمين
✓
▲
الوزن الإجمالي للشحنة — يُعتمد لتسعير الشحن الجوي والسريع (السعر عادة لكل كيلوغرام).
الحجم بالمتر المكعب — يُعتمد لتسعير الشحن البحري (السعر عادة لكل CBM، أو بحسب "الوزن الحجمي" إن كان أكبر من الوزن الفعلي).
المبلغ الذي يتقاضاه الناقل (Freight) مقابل نقل البضاعة فعلياً من الصين إلى ميناء الدخول بالمغرب.
تأمين البضاعة ضد الفقدان أو التلف أثناء النقل — عادة ما بين 0.3% و1% من قيمة البضاعة.
تكاليف إعداد مستندات الشحن (سند الشحن Bill of Lading، الفاتورة، قائمة التعبئة، إلخ).
عمولة الوسيط اللوجستي أو وكيل الشحن (Freight Forwarder) مقابل تنسيق العملية.
أي رسوم طارئة أو موسمية (رسوم الوقود، رسوم الذروة، رسوم تخزين مؤقت في ميناء الشحن، إلخ).
طريقة الحساب: إجمالي الشحن الدولي = تكلفة الشحن + التأمين + رسوم الوثائق + رسوم شركة الشحن + رسوم إضافية. (الوزن والحجم لا يُضافان مباشرة للمجموع، بل يُستخدمان كمرجع لتحديد "تكلفة الشحن" حسب نوع الشحن المختار).

الجمارك المغربية

الرسوم الجمركية، TVA ورسوم الميناء
✓
▲
نسبة "الرسم الجمركي" (Droit d'Importation) حسب كود التعريفة الجمركية (HS Code) الخاص بالمنتج.
ضريبة القيمة المضافة على الاستيراد — غالباً 20% في المغرب لمعظم المنتجات.
أتعاب المصرّح الجمركي المعتمد (العبّار) مقابل إنجاز إجراءات التخليص.
رسوم مناولة وتفريغ الحاويات/الطرود في الميناء أو المطار.
تكلفة تخزين البضاعة داخل الميناء أثناء انتظار التخليص.
رسوم الفحص الجمركي أو المخبري إن طُلب (مثل الفحص الصحي أو فحص المطابقة).
أي رسوم إضافية غير مصنّفة أعلاه.
طريقة الحساب (حسب القانون الجمركي المغربي):
1) القيمة الجمركية (CIF) = قيمة المنتج بالدرهم + إجمالي الشحن الدولي.
2) الرسوم الجمركية = القيمة الجمركية × نسبة الرسوم الجمركية %.
3) وعاء TVA = القيمة الجمركية + الرسوم الجمركية، ثم TVA = هذا الوعاء × نسبة TVA % (عادة 20%).
4) إجمالي الرسوم الجمركية = الرسوم الجمركية + TVA + (التخليص + الميناء + التخزين + الفحص + أخرى).
